2. Welche Daten wir erheben

2.1 Lokal gespeicherte Daten (kein Cloud-Backup)

Die folgenden Daten werden ausschließlich lokal auf deinem Gerät gespeichert und nicht an externe Server übertragen, solange du kein Cloud Backup aktivierst:

Datenkategorie Beispiele Zweck
Spielfortschritt Level, XP, Coins, Streak App-Kernfunktion
Quest-Daten Abgeschlossene Quests, Kalenderdaten Fortschrittsverfolgung
Körpermessungen Gewicht, Umfänge (optional eingegeben) Persönliches Tracking
App-Einstellungen Sound, Vibration, Tagesziel Wasser Personalisierung
Trial-Status Startdatum der kostenlosen Testphase Lizenzverwaltung

2.2 Daten bei Cloud Backup (optional)

Wenn du das freiwillige Cloud Backup aktivierst, meldest du dich mit deinem Google-Konto an. Dabei werden folgende zusätzliche Daten verarbeitet:

  • Google-Konto-ID (eindeutige Nutzer-ID von Google)
  • Name und E-Mail-Adresse deines Google-Kontos
  • Alle lokal gespeicherten Daten aus Abschnitt 2.1 (verschlüsselt in der Cloud)

Das Cloud Backup ist vollständig optional. Die App funktioniert vollständig ohne es. Du kannst es jederzeit deaktivieren und alle gespeicherten Cloud-Daten löschen.

2.3 Daten bei In-App-Käufen (optional)

Wenn du die Pro-Version kaufst, wird die Transaktion über den Google Play Store abgewickelt. Wir erhalten dabei lediglich eine Bestätigung des Kaufs — keine Zahlungsdaten.

5. Speicherung & Löschung

Lokale Daten

Alle lokalen App-Daten werden in SharedPreferences deines Geräts gespeichert. Sie werden gelöscht, wenn du:

  • Die App deinstallierst
  • Den App-Speicher in den Geräteeinstellungen leerst

Cloud-Daten (bei aktiviertem Backup)

Cloud-Daten werden in Firebase Firestore gespeichert und bleiben erhalten, bis du sie aktiv löschst. Du kannst deine Cloud-Daten jederzeit in der App unter Cloud Backup → Alle Daten löschen entfernen.

Nach der Löschung werden alle deine Daten innerhalb von 30 Tagen vollständig aus den Firebase-Systemen entfernt.
